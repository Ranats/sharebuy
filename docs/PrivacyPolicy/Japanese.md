---

**1. 取得する情報**

本アプリでは，以下の情報を取得します．

- 認証情報
  - Google アカウントのユーザーID，表示名，メールアドレス，プロフィール画像
  - 匿名認証の場合は匿名ID
- ユーザーが入力した情報
  - グループ情報
  - 買い物リスト，アイテム情報
- 利用状況・技術情報
  - アプリの利用状況
  - クラッシュ情報
  - 端末情報（OS種別など）

**2. 利用目的**

取得した情報は，以下の目的で利用します．

  1. 本アプリの機能提供およびデータ同期のため
  2. 本アプリの品質向上，不具合調査・修正のため
  3. 利用状況の分析およびサービス改善のため

**3. 外部サービスの利用**

本アプリは，以下の外部サービスを利用します．
- Google Firebase
  - Firebase Authentication
  - Cloud Firestore
  - Firebase Analytics
  - Firebase Crashlytics

これらのサービスにより，データは Google のサーバーに保存・処理されます．

**4. 第三者提供**

提供者は，法令に基づく場合を除き，ユーザーの同意なく個人情報を第三者に提供しません．

本アプリでは，将来的に広告配信または有料機能を導入する可能性があります．その際は，本ポリシーを適切に改定します．
